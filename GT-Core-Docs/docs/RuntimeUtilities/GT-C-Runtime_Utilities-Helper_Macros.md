# 🧩 Runtime Utilities — Helper Macros

???+ info "Short Description"

    Helper macros that standardize logging, platform checks, version checks, and extension/connection behaviors.

## 📦 Macro Groups

* **GorgeousConnectionHelperMacros** – Connection-oriented helper macros.
* **GorgeousExtensionHelperMacros** – Extension feature toggles and optional integration helpers.
* **GorgeousLoggingHelperMacros** – Logging and category helpers.
* **GorgeousPlatformHelperMacros** – Platform-specific checks.
* **GorgeousProfilingHelperMacros** – Profiling toggles and scoped instrumentation helpers.
* **GorgeousVersionHelperMacros** – Engine version checks and compatibility macros.

## 🧭 Notes

* Macro headers are located in Source/GorgeousCoreRuntimeUtilities/Public/Helpers/Macros.
* These macros are intended for C++ usage in runtime modules and tools.
