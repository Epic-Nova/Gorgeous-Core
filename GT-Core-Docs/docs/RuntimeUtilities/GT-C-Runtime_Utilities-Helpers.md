# 🧰 Runtime Utilities — Helpers

???+ info "Short Description"

    Helper classes for common runtime tasks such as file/path handling, plugin discovery, logging, and localization.

## 📁 Helper Classes

* **GorgeousConversationHelper** – Path helpers related to plugin/module locations.
* **GorgeousDatabaseHelper** – Utility functions for database-style access patterns.
* **GorgeousDirectoryHelper** – Directory creation, validation, and path normalization.
* **GorgeousFileHelper** – File IO helpers and safe path handling.
* **GorgeousLanguageHelper** – Localization and language utilities.
* **GorgeousLicenseHelper** – License file discovery and validation helpers.
* **GorgeousLoggingHelper** – Logging helpers and category utilities.
* **GorgeousPluginHelper** – Plugin presence checks and metadata access.

## 🧭 Notes

* Helpers are available under Source/GorgeousCoreRuntimeUtilities/Public/Helpers.
* Most helpers are C++-oriented but can be wrapped by Blueprint libraries where needed.
