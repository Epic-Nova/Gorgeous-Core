# 📚 Runtime Utilities — Blueprint Libraries

???+ info "Short Description"

    Blueprint function libraries that expose logging and audio utilities to Blueprint graphs.

## 🧩 Libraries

* **UGorgeousLoggingBlueprintFunctionLibrary** – Logging helpers (levels, on-screen output, categories).
* **UGorgeousAudioBlueprintFunctionLibrary** – Audio-related helper functions.

## 🧭 Notes

* Located in Source/GorgeousCoreRuntimeUtilities/Public/Libraries.
* Designed to mirror the C++ helpers with Blueprint-friendly entry points.
